# ZenQuotes

### Video Demo: https://youtu.be/D_6RODvmiGM

### Description: 
Use Public API ZenQuotes to Generate Quote of The Day

### How To Use?
1. Run ```npm i``` in the project directory
2. Run ```node index.js``` or ```nodemon index.js```
3. Navigate to https://localhost:3000/
